Tecnologias Utilizadas

  • Python 3.8+ (venv)
  • Front-end: HTML5, CSS3, JavaScript, Streamlit, CustomTkinter
  • Dados: pandas, numpy, requests, tweepy, praw, google-api-python-client
  • OCR & Criptografia: pytesseract, cryptography.Fernet, OpenCV
  • Reconhecimento Facial: face_recognition
  • PLN: transformers (BERT), spaCy, python-bcpf
  • Visualização: Dash, Plotly, matplotlib, seaborn

source .venv/bin/activate
pip install -r requirements.txt
