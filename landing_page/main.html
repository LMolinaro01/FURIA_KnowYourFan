Tecnologias Utilizadas

  • Python 3.8+ (venv)
  • Front-end: HTML5, CSS3, JavaScript, Streamlit, CustomTkinter
  • Dados: pandas, numpy, requests, tweepy, praw, google-api-python-client
  • OCR & Criptografia: pytesseract, cryptography.Fernet, OpenCV
  • Reconhecimento Facial: face_recognition
  • PLN: transformers (BERT), spaCy, python-bcpf
  • Visualização: Dash, Plotly, matplotlib, seaborn

Manual de Instalação Local

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
# pip install pandas streamlit pytesseract face_recognition ...

Instale o Tesseract OCR conforme documentação oficial para seu OS.

Obtenção de Chaves de API

  • Twitter: developer.twitter.com → App → Keys and Tokens
  • YouTube: console.cloud.google.com → APIs & Services → Credentials
